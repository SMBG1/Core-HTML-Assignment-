9.Most Common Text Formatting Tags in HTML

  1. Italic tag
    • The HTML element defines a part of text in an alternate voice or mood.The content inside is typically displayed in italic. Tip: The tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc
  2. Underline tag
    • text with the element, but this should generally not be used. The element is meant for specific use cases like marking up misspelled words, denoting proper names in Chinese text, or indicating family names
  3. Bold
    • The Bring Attention To element. The HTML element is used to draw the reader's attention to the element's contents
  4. Doctype HTML
    • The HTML DOCTYPE declaration is not an HTML element or tag. It is an instruction that tells the browser what type of document to expect. All HTML documents need to start with a
  5. Smaller text tag
    • The HTML tag reduces the font size of the text in the page by a unit
  6. Subscript tag
    • The tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font
  7. Deleted text
    • This tag in HTML stands for delete and is used to mark a portion of text which has been deleted from the document
  8. Paragraph tag
